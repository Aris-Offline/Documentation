## 什么是一个 Minecraft 皮肤

首先，让我们了解一下皮肤的本质是什么。

皮肤实际上是 PNG 纹理图像。另一方面，纹理可以理解为我们通常穿的衣服，而我们自己就是角色的主体。当我们穿着不同风格的衣服时，我们也会被不同的风格和外表所装饰。皮肤的纹理定义了我们在游戏中的风格和外观。

![](what-character-is-made-of.png)

## 视觉魔术——分辨率

皮肤的纹理赋予它风格，分辨率在决定皮肤的精细度方面起着关键作用！

如今， 64x64 大小被认为是最基本和最广泛使用的官方纹理规范，而 128x128 和 256x256 被视为高像素分辨率纹理规范。皮肤像素分辨率大小的不同会提高视觉效果"高清"的准确性，因此像素的分辨率是决定纹理是否清晰细致的重要因素之一。

![](same-skin-in-different-resolutions.png)

在 Minecraft Bedrock Edition 中，皮肤的分辨率可以是普通皮肤的两倍。这意味着皮肤模板最多可以达到 **128×128** 而不仅是 64×64 大小。普通模板仍然适用，但玩家可以使用 4 倍的像素数，因为两个维度 (图像长宽) 都增加了一倍。然而，在当前阶段的 Minecraft Java Edition 中，不支持 128x128 大小的皮肤。

## 不同的皮肤的模型

在 Minecraft 中，我们有两种模型可供选择： Steve 样式和 Alex 样式。

其主要区别在于其手臂的宽度，玩家可以根据他们设定的角色的风格进行选择。 Steve 模型的手臂较粗，而后者的手臂较细。

![](steve-model-and-alex-model.png)

在设计角色皮肤时，我们还需决定使用单层模型还是双层模型。

**单层模型 (通常为 64×32 或更高):** 这意味着只有一层角色皮肤，手脚只能设计一种样式，然后游戏会自动镜像到另一侧。这可能会限制你的设计空间，但如果你是第一次设计皮肤，这是很方便的。

**双层模型 (通常为 64×64 或更高)**: 在单层皮肤的基础上，增加了一层图案，就像为角色穿上外套和帽子一样。而且，双皮的手和脚可以分开设计，一侧不会镜像到另一侧。这无疑提供了跟多的创作乐趣性和选择性。通常，这是在熟练使用皮肤设计后更常用的方式。

## 总结

总之， Minecraft 皮肤是低维的 PNG 图像，可以像包纸盒一样，包裹在游戏中的角色周围。皮肤最常见的分辨率是 64 x64 像素。 Java 版本的游戏有易于制作的皮肤，而 Bedrock 在主角的身体上添加了一些深度和自定义凸起。
