# 介绍

此文档是对 iNKORE Studios 旗下的 MCSkinn 软件的简要介绍。如果您已知晓本软件的主要作用，请直接转到下一篇文档。

**MCSkinn** 是一个由 **iNKORE Studios** 推出的现代风格 Minecraft 皮肤管理/编辑器。它具有优秀且与 Windows 契合的 Fluent 设计风格，易用的可视化编辑器，以及实用的编辑工具。

皮肤是您在 Minecraft 中展现个性，提高辨识度的重要方式。通过 **MCSkinn**，您便能够自由地创建 Minecraft 皮肤。

**MCSkinn** 除了其优秀的设计和性能表现外，也具有同其他皮肤编辑器趋同的功能设计。如果您作为新手并希望能够创建自己的 Minecraft 皮肤，您可以参阅后面的文档（尤其是教程部分），这会为您的皮肤创建大有效益。

请继续阅读后面的文档。祝您借助 MCSkinn 能够创建漂亮且成功的皮肤！
