# 提示与技巧

本文包含了所有快捷键和一些有用的提示，您可以在 MCSkinn 中使用它们。巧妙地使用这些快捷键可以提高您的编辑效率。

## 快捷键

:::note

由于版本更新，MCSkinn 的快捷键可能会有所变化。以下内容适用于 Classic 版本。

:::

- 新建皮肤 `Ctrl + N`

- 撤销 `Ctrl + Z`

- 重做 `Ctrl + Y`

- 保存 `Ctrl + A`

- 另存为 `Ctrl + Shift + S`

- 全部保存 `Ctrl + Alt + S`

- 删除 `Ctrl + D` `Delete`

- 透视图（3D 视图） `Numpad 1`

- 纹理图（2D 视图） `Numpad 2`

- 分割视图 `Numpad 3`

- 切换视图模式 `Tab`

- 切换地面 `Ctrl + G`

- 切换幽灵部件 `Ctrl + Shift + G`

- 手 `H`

- 铅笔 `P` `B`

- 橡皮擦 `E`

- 吸管 `I`

- 减淡/加深 `O`

- 变暗/变亮 `D`

- 填充桶 `G`

- 噪声 `N`

- 图章 `S`
