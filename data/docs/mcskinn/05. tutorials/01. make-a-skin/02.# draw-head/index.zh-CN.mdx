# 绘制头部

画出头部是您开始绘制皮肤的第一步骤，这对于皮肤定型有很大作用。下面将详细讲解如何绘制出一个头部。

## 内层头发

对于 64x64 的一般皮肤，头部区域只有 8×8 。这种有限尺寸下无法画出大量细节，所以我们需要充分利用空间。

在绘制头部前，最好隐藏其他部位，以便观察头部。在 *部位* 面板，取消选中所有除 "Head" 之外的其他部位，这会有助于在 3D 视图下查看头部。

绘制头部的重要前提就是选择皮肤和头发的颜色。在调色板调好之后，先使用肤色填充整个头部。

填充好之后，使用发色绘制头发的轮廓和内侧。

:::tip

发型画法有一定限制。如果您找不到灵感，您可以借鉴其他人的发型画法。您也可以通过刘海来增加皮肤的质感。

:::

内层的刘海只需大致轮廓，稍后我们会绘制外层刘海，所以无需太担心现在看起来死板的刘海。

## 添加细节

为了给头发添加更多细节，您需要使用 *变亮/变暗* 工具。请在工具栏中选中它。

首先需要在头发中添加阴影。在右侧边栏中选择 *变暗*，随后在两侧头发上各绘制 3 个像素点以确定明暗分界，并根据头发的走向绘制更多细节。

:::tip

不要画太规则的纹路，最好有一定随意性。并且不要过多地使用噪点工具。

:::

接下来，我们添加一些高光。这和绘制阴影的方式类似，但是不要画过多了。

对于其他的几面，请以相似的方式绘制。

在头顶处，先绘出头发的分界线，并在分界线周围添加高光和阴影。同上文一样，最好具有一定的随机性。

最后，连接相邻面的像素，保证面与面之间看起来不脱节。

## 外层头发

您可以根据上文的方法绘制外层。为了更立体的效果，您可以不画第二层的顶面，这样您可以获得更蓬松的效果。

首先，我们需要使外层可见。在*部位*面板，并选中 *外层头部* (通常被称为 "Head Armor") 。

然后，画出外层轮廓。为了快速区分图层，可以使用更亮的颜色，在最终完成后使用填充桶来替换颜色。

接着使用上述方法，在外层添加阴影和高光。

:::tip

您可以尝试不同的发型。不要拘泥于 ~copy-and-paste~，为自己皮肤带来一些特色。



## 眼睛

眼睛有多种画法，但通常比较好看的只有几种。以下是一些样式。

![](eyes-in-mc-be.png)

*在 Minecraft: Bedrock Edition 中的一些眼睛样式*

我们建议您不要画嘴巴 (因为空间有限，而且看起来不好看) ，但在特殊情况下可以试试，这可能会使您的皮肤锦上添花。

首先为你的瞳孔和睫毛选择合适的颜色。接着画瞳孔，然后选择眼白的颜色。不要使用纯白色，瞳孔应该有轻微的颜色倾向 (S: 约 1~2，V: 约 99~98) 。

然后，使用 *减淡/加深* 工具添加更多细节。
