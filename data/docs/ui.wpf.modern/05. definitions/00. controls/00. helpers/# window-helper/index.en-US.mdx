# WindowHelper Class

Provides attached properties and style management for the window component.

## Definition

- Namespace: [iNKORE.UI.WPF.Modern.Controls.Helpers](.)
- Assembly: iNKORE.UI.WPF.Modern.dll

```csharp
public static class WindowHelper {...}
```

Inheritance: No inheritance for static classes.

## Attached Properties

### UseModernWindowStyle

Determines whether the modern style should be applied to the window.

More information on [Window](../../../components/dialogs/window#windowhelperusemodernwindowstyle).

- Type: `bool`

- Default value: `false`

### SystemBackdropType

Specifies the system backdrop of the window.

More information on [Window](../../../components/dialogs/window#windowhelpersystembackdroptype).

- Type: [BackdropType](../../helpers/styles/backdrop-type)

- Default value: `BackdropType.None`

### UseAeroBackdrop

This is obsolete and no longer works. Use **SystemBackdropType** instead.

### UseAcrylicBackdrop

This is obsolete and no longer works. Use **SystemBackdropType** instead.

### Acrylic10Color

The color of the acrylic effect when applying acrylic using the older API.

More information on [Window](../../../components/dialogs/window#windowhelperacrylic10color).

- Type: `Color`

- Default value: `Color.FromArgb(0x99, 0x00, 0x00, 0x00)`

### ApplyBackground

Determines whether the background should be applied to the window.

More information on [Window](../../../components/dialogs/window#windowhelperapplybackground).

- Type: `bool`

- Default value: `true`

### ApplyNoise

Determines whether an extra noise layer should be applied to the window.

More information on [Window](../../../components/dialogs/window#windowhelperapplynoise).

- Type: `bool`

- Default value: `false`

### CornerStyle

Determines the corner style of the window.

More information on [Window](../../../components/dialogs/window#windowhelpercornerstyle).

- Type: [WindowCornerStyle](../../helpers/styles/window-corner-style)

- Default value: `WindowCornerStyle.None`

### FixMaximizedWindow

Determines whether an extra maximize fixer should be applied to the window.

More information on [Window](../../../components/dialogs/window#windowhelperfixmaximizedwindow).

- Type: `bool`

- Default value: `true`

### UpdateWindowChrome

This method will be invoked automatically when the WindowChrome needs to be updated. You can also call this method manually to update the WindowChrome when anything unexpected happens.

```csharp
public static WindowChrome UpdateWindowChrome(this Window window);
```

## Remarks

### Modern style needs to be applied manually

Unlike other controls, the modern style will not be applied automatically. You need to set **UseModernWindowStyle** to `true` to apply the modern style to the window.
