# 下拉列表框

使用下拉列表框（也称为下拉列表）来显示用户可以选择的项目列表。下拉列表框以紧凑状态开始，并展开以显示可选择的项目列表。列表框类似于下拉列表框，但不可折叠/没有紧凑状态。您可以在本文末尾了解更多关于列表框的信息。

- **类**: [System.Windows.Controls](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls)[.ComboBox](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.combobox)

- **继承**: [Object](https://learn.microsoft.com/en-us/dotnet/api/system.object) → (...) → [Control](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.control) → [ItemsControl](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.itemscontrol) → [Selector](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.primitives.selector) → [ComboBox](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.combobox)

## 示例

以下示例创建了一个下拉列表框。该示例通过将 ItemsSource 属性绑定到 VacationSpots 类型的集合对象来填充下拉列表框。该示例还创建了一个 TextBlock 来显示下拉列表框的选定项目。

```xml
<StackPanel  Grid.Column="0"
        Grid.Row="6">
    <StackPanel.Resources>
        <local:VacationSpots x:Key="myVacations"/>
    </StackPanel.Resources>
    <ComboBox Name="comboBox1"
        ItemsSource="{StaticResource myVacations}"
        Text="My Cities"
        Width="200"
        IsEditable="true"
        IsReadOnly="true"/>

    <TextBlock Text="{Binding ElementName=comboBox1, Path=SelectedItem}" Margin="0,10,0,0" HorizontalAlignment="Center"/>
</StackPanel>
```

以下示例定义了前述示例中下拉列表框绑定的集合对象。

```csharp
class VacationSpots : ObservableCollection<string>
{
    public VacationSpots ()
    {
        Add("Spain");
        Add("France");
        Add("Peru");
        Add("Mexico");
        Add("Italy");
    }
}
```

然后您可以从下拉列表框中选择一个城市。

## 备注

### 这是正确的控件吗？

使用下拉列表让用户从一组项目中选择一个值，这些项目可以用单行文本充分表示。

使用列表或网格视图而不是下拉列表框来显示包含多行文本或图像的项目。

当项目少于五个时，考虑使用单选按钮（如果只能选择一个项目）或复选框（如果可以选择多个项目）。

当选择项目在您的应用程序流程中次要时，请使用下拉列表框。如果默认选项在大多数情况下对大多数用户是推荐的，使用列表视图显示所有项目可能会比必要的吸引更多注意力。您可以通过使用下拉列表框来节省空间并减少干扰。

:::tip 建议

将下拉列表框项目的文本内容限制为单行。
按最合逻辑的顺序对下拉列表框中的项目进行排序。将相关选项分组，并将最常见的选项放在顶部。按字母顺序对名称进行排序，按数字顺序对数字进行排序，按时间顺序对日期进行排序。

:::

## ItemsControl

下拉列表框允许用户从下拉列表中选择一个项目，或者在控件的文本框中输入新文本。下拉列表框是一个 ItemsControl，这意味着它可以包含任何类型的对象集合（例如字符串、图像或面板）。有关更多信息，请参阅 [ItemsControl](https://learn.microsoft.com/en-us/dotnet/api/system.windows.controls.itemscontrol) 类。

### 扩展属性

您可以使用 `ui:ComboBox` 的附加属性来自定义您的下拉列表框。

- **TextBoxStyle**: 获取下拉列表框中 TextBox 的样式，当下拉列表框是可编辑的。

### 提升

下拉列表框控件默认带有提升边框效果。您可以使用 `ui:ElevationBorder` 自定义其提升效果。
